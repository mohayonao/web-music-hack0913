## web-music-hack0913

### 二つの数字から音楽パターンを作ります

### 例えば 5 と 2 の場合

### 5bit で表せる 32 の数値のうち

### ビットが 2つ 立っているものを選択します

 - 00011
 - 00101
 - 00110
 - 01001
 - 01010
 - 01100
 - 10001
 - 10010
 - 10100
 - 11000

### それぞれを変換して

  - 00011 => [ 0, 0, 0, 2, 1 ]
  - 00101 => [ 0, 0, 4, 0, 1 ]

### 全部くっつける

  - [ 0, 0, 0, 2, 1, 0, 0, 4, 0, 1 ... ]

### 演奏時は時間に応じてインデックスが増加する

### データとインデックスの余りを求める

  - index: 100
  - data : 32

### だったら 100 % 32 で 4

### その番号に応じた演奏をします。

### パートは上から

  - ドラム
  - ベース
  - シーケンス
  - シンセ

です
